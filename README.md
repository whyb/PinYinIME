# 极简拼音输入法 (PinyinIME)

一个极简的 Windows 拼音输入法，纯 C++ / Win32 API 实现，零外部依赖。

## 特性

- **零依赖**: 仅使用 Windows API，无需安装任何运行时
- **内置词库**: 900+ 条目（单字、词组、简拼），编译进 exe
- **全局热键**: 右 Shift 切换中英文模式
- **智能候选**: 支持全拼、简拼、前缀匹配
- **自学习**: 自动记录用户选择，频率动态调整
- **轻量**: 单文件 ~500 行，编译后 < 200KB

## 编译

### 环境要求

- Windows 7+
- Visual Studio 2012+ (或 VS Build Tools)

### 编译步骤

1. 打开 **VS2012 Developer Command Prompt** (x64)
2. 进入项目目录
3. 运行编译脚本:

```cmd
cd D:\codes\github\InputMethod
build.bat
```

或手动编译:

```cmd
cl.exe /utf-8 /O2 /MT /EHsc /DUNICODE /D_UNICODE main.cpp /Fe:PinyinIME.exe user32.lib gdi32.lib kernel32.lib
```

## 使用方法

1. 运行 `PinyinIME.exe`
2. 按 **右 Shift** 切换中英文模式
3. 中文模式下:
   - 输入拼音字母 (a-z)
   - 按 **空格** 确认第一个候选
   - 按 **1-9** 选择对应候选
   - 按 **- / =** 翻页
   - 按 **Backspace** 删除最后一个字母
   - 按 **Escape** 清空输入
4. 再按 **右 Shift** 切回英文模式

## 文件说明

| 文件 | 说明 |
|------|------|
| `main.cpp` | 主程序 (钩子、引擎、候选框、注入) |
| `dictionary.h` | 内嵌词库 (500单字+300词组+100简拼) |
| `build.bat` | 编译脚本 |
| `user.dict` | 用户自学习词库 (运行时自动生成) |

## 词库结构

- **单字表**: 覆盖全部拼音音节的高频字
- **词组表**: 常用双字词、三字词、四字成语
- **简拼表**: 首字母缩写 (如 "nh" → "你好")

## 自学习

每次选择候选时:
1. 该候选频率 +1
2. 保存到 `user.dict` 文件
3. 下次输入相同拼音时优先显示

## 注意事项

- 需要以**管理员权限**运行才能在某些应用中生效
- 全局钩子会略微影响系统性能
- 首次运行会在同目录生成 `user.dict` 文件
- 候选框使用 GDI 绘制，外观简洁
